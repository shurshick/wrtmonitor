# v0.1.1-rc4-dark-dashboard-telemetry

## Dashboard

- WebUI переведён на Modern Dark Network Dashboard: тёмная flat-тема, контрастные статусы, компактные панели и читаемая история команд.
- Android использует согласованную тёмную Material-тему.

## Telemetry

- Agent собирает память, CPU, storage, температуру при наличии датчика, процессы, суммарный RX/TX, интерфейсы, Wi-Fi и расширенные ubus snapshots.
- WebUI и Android отображают эти сведения структурированными разделами: система, оборудование, сеть и Wi-Fi.
- Не передаются Wi-Fi-пароли, MAC-адреса клиентов или содержимое трафика.

## Agent

- WrtMonitor сам раздаёт установочные файлы агента по `/downloads/openwrt/`.
- Документация дополнена установкой с собственного сервера, обновлением и полным удалением агента.
