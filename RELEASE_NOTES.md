# v0.10.3

Исправление бесконечного роста журнала команд и его отображения в Web UI.

## Изменения

- Журнал операций показывает пять компактных записей на страницу.
- Добавлена постраничная навигация без загрузки всей истории.
- Завершённые команды автоматически удаляются через 30 дней.
- Дополнительно сохраняется не более 500 завершённых команд на роутер.
- Активные команды `queued`, `sent` и `running` очистка не затрагивает.
- Срок и лимит настраиваются через переменные окружения и TrueNAS YAML.
- Скачивание резервных копий и диагностических архивов не зависит от открытой страницы журнала.
- Android `versionCode` увеличен до `44`.

## Обновление

Обновите сервер через TrueNAS redeploy: **Apps -> WrtMonitor -> Edit -> Save**.
Миграция БД не требуется. Очистка старых записей выполнится автоматически при
работе с журналом и при получении результатов команд от агента.
