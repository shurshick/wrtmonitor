# v0.11.0

Новая структура управления роутером в Web UI и Android.

## Изменения

- Web UI получил группировку разделов по задачам: состояние, подключения, домашняя сеть, безопасность и управление.
- Firewall, NAT, статические маршруты и UPnP вынесены из общей страницы интернета в отдельные «Сетевые правила».
- VPN отделён от обычных WAN-настроек и доступен как самостоятельный раздел.
- На обзоре роутера добавлены быстрые переходы к сетевым правилам и VPN со сводными счётчиками.
- Android получил пять основных вкладок: обзор, клиенты, Wi-Fi, интернет и «Ещё».
- Редкие и служебные функции Android собраны в понятный каталог, а сетевые правила и VPN открываются отдельными экранами.
- Переходы назад в Android теперь сохраняют иерархию разделов, а не выбрасывают пользователя к списку роутеров.
- Управляющие элементы по-прежнему скрываются по фактическим capabilities подключённого OpenWrt.
- Исправлен compatibility-переход агента `0.9.0`: обновление на `0.11.0` больше не блокируется как downgrade.
- Android `versionCode` увеличен до `46`.

## Обновление

Обновите сервер через TrueNAS redeploy: **Apps -> WrtMonitor -> Edit -> Save**.
Миграция БД не требуется. Агент может обновиться штатно из раздела «Обслуживание».
