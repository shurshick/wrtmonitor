# v0.5.0

Релиз управления клиентами домашней сети.

## Изменения

- Добавлен постоянный реестр клиентов с именами, vendor lookup, статусом и историей RX/TX.
- Появились индивидуальные и групповые профили: блокировка, расписание и приоритет трафика.
- Web UI и Android получили одинаковое управление клиентами, профилями и SQM.
- Агент собирает счётчики клиентов через `nlbwmon`; зависимость устанавливается как опциональная.
- Добавлена транзакционная команда `qos.set_sqm` для общего лимита и снижения задержек канала.
- Capability report обновлён до schema v6; Android `versionCode` увеличен до `36`.

## Обновление

- Сервер: выполните redeploy `ghcr.io/shurshick/wrtmonitor:latest`.
- Агент: обновите до `0.5.0`; `nlbwmon` будет установлен, если пакет есть в feed роутера.
- Android: установите `wrtmonitor-android-v0.5.0-debug.apk` поверх предыдущей версии.

PostgreSQL volume при обновлении контейнера сохраняется. Alembic создаст таблицы реестра клиентов.
