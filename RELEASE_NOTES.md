# v0.13.7

Исправление недостоверных показателей клиентов, трафика и Wi-Fi эфира.

## Изменения

- В обзоре и Android показывается число реально активных клиентов, а не размер накопленного реестра.
- Сохранённые и отключённые клиенты остаются в разделе домашней сети и не искажают оперативную статистику.
- Wi-Fi эфир показывает только полученные от драйвера значения: сигнал, скорость RX/TX, длительность подключения и airtime RX/TX.
- Сырые словари hostapd и пустые декоративные колонки удалены.
- Трафик клиентов отображается только при рабочем nlbwmon и наличии фактических счётчиков.
- Проверка capability `telemetry.clients.traffic` теперь обращается к базе nlbwmon, а не только ищет бинарник.
- Android `versionCode` повышен до `55`.

## Обновление

Сервер обновляется через TrueNAS redeploy: **Apps -> WrtMonitor -> Edit -> Save**. Миграций БД нет. После обновления сервера агент автоматически обновится до `0.13.7`; актуальные показатели появятся после следующей telemetry.
