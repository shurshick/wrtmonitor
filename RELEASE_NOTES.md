# v0.13.6

Поддержка OpenWrt 25.12: пакеты `apk` и исправленное определение резервных копий.

## Изменения

- Агент определяет `apk` на OpenWrt 25.12+ и `opkg` на предыдущих версиях.
- Просмотр, обновление каталога, установка и удаление пакетов работают через обнаруженный менеджер.
- Web UI показывает фактический менеджер пакетов роутера.
- Резервная копия доступна при наличии `sysupgrade`, `tar` и `base64`; отдельный `gzip` больше не требуется.
- Добавлен полный системный `PATH` для корректного обнаружения `sysupgrade` из фонового сервиса.
- Capability schema повышена до `13`, Android `versionCode` — до `54`.

## Обновление

Сервер обновляется через TrueNAS redeploy: **Apps -> WrtMonitor -> Edit -> Save**. Миграций БД нет. После обновления сервера агент автоматически обновится до `0.13.6`; затем пакеты и резервная копия станут доступны после следующей telemetry.
