# v0.13.2

Компактный и управляемый список клиентов домашней сети в Web UI и Android.

## Изменения

- Крупные плитки клиентов заменены компактными строками с одинаковыми колонками.
- По нажатию строка раскрывает параметры соединения, трафик и управление политикой доступа.
- Добавлены поиск по имени/IP/MAC и фильтры онлайн/офлайн.
- Android использует ту же структуру: общий список и одна раскрытая панель клиента.
- На мобильной ширине скрываются второстепенные колонки, основные сведения остаются видимыми.
- Browser E2E проверяет поиск, фильтры, раскрытие и отсутствие горизонтального переполнения.
- Android `versionCode` увеличен до `50`.

## Обновление

Обновите сервер через TrueNAS redeploy: **Apps -> WrtMonitor -> Edit -> Save**. Миграций БД в этом выпуске нет. Android можно установить поверх предыдущей версии. Агент обновляется штатно из раздела «Обслуживание».
