# v0.13.1

Исправление и расширение монитора telemetry в Web UI и Android.

## Изменения

- Устранена гонка запросов истории в Android: поздний ответ старого диапазона больше не затирает новый.
- Переключатели показывают выбранный и фактически загруженный период, загрузку и ошибку API.
- Web UI и Android получили шкалы значений и времени, легенду и фактическое покрытие данных.
- График переключается между трафиком, нагрузкой, памятью и количеством клиентов.
- Web UI защищён от поздних ответов при быстром переключении 2 часа / 24 часа / 7 дней / 30 дней.
- Android-разделы «Система» и «Обслуживание» разделены; пакеты, прошивка, backup и агент больше не смешаны с ресурсами ОС.
- Browser E2E проверяет реальную загрузку диапазона 24 часа и процентную шкалу памяти.
- Android `versionCode` увеличен до `49`.

## Обновление

Обновите сервер через TrueNAS redeploy: **Apps -> WrtMonitor -> Edit -> Save**. Миграций БД в этом выпуске нет. Android можно установить поверх предыдущей версии. Агент обновляется штатно из раздела «Обслуживание».
