# v0.3.0-rc3

Релиз очистки данных и переработки основных рабочих разделов.

## Что изменилось

- Роутер теперь можно удалить независимо от его текущего статуса.
- Удаление безвозвратно очищает запись устройства, telemetry, очередь и историю команд, а также связанные audit-записи.
- Старый device token сразу становится недействительным; для повторного подключения агент нужно зарегистрировать заново.
- Web UI и Android используют новый `DELETE /api/v1/devices/{device_id}`.
- Раздел «Система» получил понятные показатели памяти, накопителя, нагрузки, температуры, conntrack, uptime и состояния служб.
- Интернет, домашняя сеть и Wi-Fi разделены на обзор состояния, основные действия и раскрываемые расширенные настройки.
- Байты и килобайты отображаются в читаемых единицах.
- Убраны технические подписи `risk: level N`, формы и кнопки стали компактнее.
- Web UI проверен на разрешениях `1280x720` и `390x844` без горизонтального переполнения.
- Android `versionCode` увеличен до `30`.

## Обновление

Сервер: выполните redeploy образа `ghcr.io/shurshick/wrtmonitor:latest`. PostgreSQL volume сохраняется.

Android: установите `wrtmonitor-android-v0.3.0-rc3-debug.apk` поверх предыдущей версии.

OpenWrt agent можно обновить штатной кнопкой или автообновлением до `0.3.0-rc3`.
