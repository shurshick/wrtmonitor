# v0.13.4

Полная переработка списка клиентов Android и исправление определения активных устройств.

## Изменения

- Android показывает клиентов плотным списком вместо вложенных технических карточек.
- Клиенты сгруппированы по Wi-Fi-сети, проводному подключению и доступности; недоступные устройства свёрнуты по умолчанию.
- Поиск и фильтры работают по имени, IPv4, MAC, производителю и SSID.
- Параметры выбранного клиента открываются на отдельном экране с нормальной системной навигацией назад.
- Настройки клиента разделены на основные параметры, профиль, расписание, приоритет, лимиты и постоянный IPv4.
- IPv4 имеет приоритет над служебными IPv6-адресами в списке и карточке клиента.
- Агент передаёт SSID, диапазон и интерфейс активных Wi-Fi-клиентов.
- DHCP-аренда и старая статистика трафика больше не считаются признаком нахождения клиента в сети.
- Capability schema обновлена до `12`, Android `versionCode` — до `52`.

## Обновление

Сервер обновляется через TrueNAS redeploy: **Apps -> WrtMonitor -> Edit -> Save**. Миграций БД нет. После обновления сервера обновите агент до `0.13.4`, чтобы Android получил SSID и диапазон активных Wi-Fi-клиентов. APK устанавливается поверх предыдущей версии.
