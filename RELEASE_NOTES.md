# v0.13.3

Управление клиентами, межсетевым экраном и обслуживанием роутера без ручных идентификаторов и заглушек.

## Изменения

- Постоянный IPv4 перенесён в параметры конкретного клиента; текущий DHCP-адрес можно закрепить одной кнопкой.
- Мобильный список клиентов сгруппирован по сетевым интерфейсам и получил компактные строки с раскрываемыми параметрами.
- Профили доступа в Web UI получили аккуратную отдельную форму создания.
- Правила firewall, зоны и транзит теперь редактируются и удаляются по реальным UCI section в Web UI и Android.
- Для пакетов opkg добавлены обновление каталога, установка, обновление и безопасное удаление.
- Для резервных копий добавлены создание, скачивание и восстановление из файла.
- Агент передаёт списки установленных и доступных к обновлению пакетов, а также UCI section объектов firewall.
- Убраны пустые вертикальные разрывы между панелями настроек на широких экранах.
- Capability schema обновлена до `11`, Android `versionCode` — до `51`.

## Обновление

Сервер обновляется через TrueNAS redeploy: **Apps -> WrtMonitor -> Edit -> Save**. Миграций БД нет. После обновления сервера обновите агент до `0.13.3`, чтобы получить списки пакетов и управление существующими объектами firewall. Android APK устанавливается поверх предыдущей версии.
