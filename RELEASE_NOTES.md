# v0.10.2

Исправление фактических параметров роутера в формах управления.

## Изменения

- Агент передаёт реальный IPv4 LAN, длину префикса, маску, DHCP-пул, часовой пояс и NTP-настройки.
- Сервер понимает новый и прежний форматы сетевой telemetry и больше не теряет IPv4, IPv6, gateway и DNS.
- Web UI и Android заполняют LAN/WAN/DHCP/время фактическими значениями роутера вместо демонстрационных значений.
- Маски сети, протокол WAN, интерфейсы, Wi-Fi-каналы, регион, режим, часовой пояс, дни недели и другие перечислимые параметры переведены на выбор из списка.
- Удалены подставные адреса, интерфейсы, DDNS-провайдер и параметры WireGuard из действующих форм.
- Android получает firewall-зоны с роутера, а агент выбирает для гостевого Wi-Fi свободную подсеть вместо фиксированного адреса.
- Добавлены регрессионные тесты LAN `192.168.31.1/24`, старого формата `ubus` и настроек времени.
- Android `versionCode` увеличен до `43`.

## Обновление

1. Обновите сервер через TrueNAS redeploy: **Apps -> WrtMonitor -> Edit -> Save**.
2. После появления версии сервера `0.10.2` обновите агент в разделе **Обслуживание**.
3. После новой telemetry формы LAN, DHCP и времени заполнятся реальными параметрами роутера.
