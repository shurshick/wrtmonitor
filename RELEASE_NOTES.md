# v0.3.0-rc5

Релиз аккуратных и компактных действий в Web UI и Android.

## Что изменилось

- Кнопки получили единую высоту, радиусы, отступы и размер текста.
- Основные действия стали компактнее, вторичные оформлены контуром.
- Опасные команды больше не выглядят как крупные красные плашки.
- В Web UI убрано растягивание кнопок обслуживания на всю ширину карточки.
- На мобильной ширине сетевые действия сохраняют естественный размер.
- Android использует один компонент действий на всех экранах, включая авторизацию и настройки.
- Видимая высота Android-кнопки уменьшена до 34 dp при сохранении удобной области касания.
- Android `versionCode` увеличен до `32`.

## Обновление

Установите `wrtmonitor-android-v0.3.0-rc5-debug.apk` поверх предыдущей версии.

Для обновления Web UI выполните redeploy образа `ghcr.io/shurshick/wrtmonitor:latest`. PostgreSQL volume сохраняется.
