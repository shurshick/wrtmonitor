# v0.3.0-rc2

Исправляющий релиз раздела «Обслуживание».

## Что изменилось

- Убрана единая длинная карточка агента, в которой были смешаны состояние, обновление, интервал и диагностика.
- Добавлена компактная строка состояния с версией агента, версией сервера и интервалом связи.
- Обновление агента, обмен данными и диагностика разделены на самостоятельные рабочие блоки.
- Для устаревшего агента выводится заметное предупреждение и кнопка обновления.
- Rollback показывается только при наличии резервной версии.
- Результат диагностики находится внутри диагностического блока.
- История команд оформлена как отдельный журнал операций.
- Capabilities и raw telemetry убраны в сворачиваемый блок «Технические данные».
- Сохранена отдельная опасная зона отключения роутера.
- Android `versionCode` увеличен до `29`.

## Обновление

Сервер: выполните redeploy образа `ghcr.io/shurshick/wrtmonitor:latest`. PostgreSQL volume сохраняется.

Android: установите `wrtmonitor-android-v0.3.0-rc2-debug.apk` поверх предыдущей версии.

OpenWrt agent можно обновить штатной кнопкой или автообновлением до `0.3.0-rc2`.
