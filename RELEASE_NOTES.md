# v0.12.0

Живой визуальный слой Web UI и Android.

## Изменения

- Обзор роутера в Web UI полностью перестроен в рабочий сетевой dashboard.
- Добавлен живой график входящей и исходящей скорости с интервалами 5 минут, 30 минут и 2 часа.
- Добавлены индикаторы памяти, нагрузки, клиентов, WAN, Wi-Fi и быстрые переходы к управлению.
- Dashboard обновляет данные каждые 5 секунд без перезагрузки страницы.
- Android получил тот же live traffic monitor, ресурсную сводку и фоновое обновление.
- Добавлен защищённый API истории telemetry с расчётом скорости по накопительным счётчикам.
- Адаптивный E2E теперь проверяет desktop/mobile layout и фактическую отрисовку canvas.
- Иконка Android приведена к актуальной тёмной сетевой палитре.
- Android `versionCode` увеличен до `47`.

## Обновление

Обновите сервер через TrueNAS redeploy: **Apps -> WrtMonitor -> Edit -> Save**. Миграция БД не требуется. Агент обновляется штатно из раздела «Обслуживание».
