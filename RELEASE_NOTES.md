# v0.13.5

Компактные раскрываемые настройки Web UI и актуальные скриншоты проекта.

## Изменения

- На страницу проекта добавлены актуальные скриншоты обзора, домашней сети и интернет-интерфейсов.
- «Межсетевой экран» и «Зоны и транзит» свёрнуты при открытии страницы.
- Идентификация роутера и настройка времени открываются по нажатию.
- Статус и перезапуск системных служб собраны в одной раскрываемой секции.
- Browser E2E проверяет закрытое состояние и раскрытие секций на desktop и mobile.
- Android `versionCode` повышен до `53`.

## Обновление

Сервер обновляется через TrueNAS redeploy: **Apps -> WrtMonitor -> Edit -> Save**. Миграций БД нет. APK устанавливается поверх предыдущей версии. Изменений протокола агента нет.
