# v0.13.0

Долговременная telemetry и единое управление Wi-Fi.

## Изменения

- Добавлен компактный ряд telemetry-метрик с хранением по умолчанию 45 дней.
- Web UI и Android получили реальные диапазоны графика: 2 часа, 24 часа, 7 и 30 дней.
- Сервер выполняет downsampling длинных диапазонов и не передаёт клиентам десятки тысяч точек.
- Latest telemetry теперь содержит предупреждения о потере связи, высокой загрузке памяти и недоступном WAN.
- Настройки Wi-Fi в Web UI собраны в один блок радиомодуля и одну карточку на SSID.
- Переключение `radio0/radio1` подставляет фактические channel, htmode, country и txpower выбранного модуля.
- В Android добавлен явный выбор радиомодуля; параметры 2.4/5 ГГц больше не смешиваются.
- Удалены дублирующие формы канала, региона, SSID и пароля.
- Добавлена миграция PostgreSQL `0006_telemetry_metrics`.
- Android `versionCode` увеличен до `48`.

## Обновление

Обновите сервер через TrueNAS redeploy: **Apps -> WrtMonitor -> Edit -> Save**. Alembic автоматически создаст таблицу долговременных метрик, существующие данные и настройки не удаляются. Агент обновляется штатно из раздела «Обслуживание».
