# Android-приложение

Android-приложение — основной интерфейс пользователя.

Целевые экраны:

- первая настройка сервера;
- вход администратора;
- список роутеров;
- состояние роутера;
- Wi-Fi;
- сеть;
- клиенты;
- системные действия;
- журнал событий;
- настройки.

Интерфейс строится на Material 3.
