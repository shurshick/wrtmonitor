# Android-приложение

Android-приложение — основной интерфейс пользователя.

Текущий каркас построен на Kotlin, Jetpack Compose и Material 3. Интерфейс поддерживает русский и английский языки.

При первом запуске приложение проверяет сохранённый адрес сервера. Если адрес ещё не задан, открывается стартовый экран настройки с полем `Адрес сервера`. Адрес сохраняется локально в настройках приложения и позже может быть изменён в разделе `Настройки`.

Целевые разделы:

- первая настройка сервера;
- вход администратора;
- список роутеров;
- состояние роутера;
- Wi-Fi;
- сеть;
- клиенты;
- системные действия;
- журнал событий;
- настройки.

На старте в репозитории есть scaffold с основными вкладками. Следующий шаг — подключение API-клиента и сохранение токена входа на устройстве.
