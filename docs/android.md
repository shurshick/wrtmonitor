# Android-приложение

Android-приложение — основной интерфейс пользователя.

Текущий каркас построен на Kotlin, Jetpack Compose и Material 3. Интерфейс должен поддерживать русский и английский языки.

Целевые разделы:

- первая настройка сервера;
- вход администратора;
- список роутеров;
- состояние роутера;
- Wi-Fi;
- сеть;
- клиенты;
- системные действия;
- журнал событий;
- настройки.

На старте в репозитории есть scaffold с основными вкладками. Следующий шаг — подключение API-клиента и сохранение адреса сервера/токена на устройстве.
